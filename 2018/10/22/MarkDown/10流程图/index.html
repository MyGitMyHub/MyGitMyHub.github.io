可用模块

  1. 开始 start
    st=>start: 开始
  2. 结束 end
    e=>end: 结束
  3. 普通操作块 opration
    op1=>opration: 第一个操作
    op2=>opration: 第二个操作
  4. 判断块 condition
    cond1=>condition: 第一个判断
    cond2=>condition: 第二个判断
  5. 输入输出块 inputoutput
    io1=>inputoutput: 输入输出块1
    io2=>inputoutput: 输入输出块2
  6. 子任务块
    sub1=>subroutine: 子任务1
    sub2=>subroutine: 子任务2

判断与位置控制

  1. 判断流程控制
    con1(yes)->op1 #yes 的时候回到 op1
    con1(no)->e #no 的时候到结束
  2. 位置指定
    cond1(no)->op2(right)->op1 #控制 op2 位置置于右边，再由op2 返回 op1
    #还可以这样 cond1(no,right)
    cond1(yes)->e
